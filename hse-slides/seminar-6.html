equals и hashCode

equals

equals позволяет определить, являются ли два объекта эквивалентными.

Например, логично что следующие вызовы вернут true:

  • Integer.valueOf(1).equals(Integer.valueOf(1))

  • "abc".equals("abc")

  • new City("Moscow").equals(new City("Moscow"));

Контракт equals

  1. Рефлексивность

  2. Симметричность

  3. Транзитивность

  4. Согласованность (consistency)

  5. ∀ reference x: x.equals(null) → false

Отношение эквивалентности

Рефлексивность

∀ reference типа:

  • если x != null

  • то x.equals(x) == true

Симметричность

∀ reference типов x и y, отличных от null:

  • x.equals(y) == true тогда и только тогда, когда

  • y.equals(x) == true

Транзитивность

∀ reference типов x, y, и z, отличных от null:

  • если x.equals(y) == true и

  • y.equals(z) == true, тогда

  • x.equals(z) == true

Согласованность (consistency)

"Если объект не поменялся, то equals должен возвращать тот же результат"

Рецепт

идеального equals:

Из К. Хорстманна (5.2.2., the equals method)

  1. Назвать параметр otherObject

  2. Сравнить его с this, используя ==

  3. Проверить otherObject на null

  4. Сравнить классы (getClass())

  5. Привести otherObject к типу данного класса

  6. Сравнить по полям

Рецепт (пример)

public class Point {
    private int x;
    private int y;

    @Override
    public boolean equals(Object otherObject) { // 1
        if (otherObject == this) return true; // 2
        if (otherObject == null) return false; // 3
        if (otherObject.getClass() != Point.class) return false; // 4
        Point other = (Point)otherObject; // 5
        return x == other.x && y == other.y; // 6
    }
}

Контракт hashCode

  • Согласованность

  • Если x.equals(y) == true, то:

    • x.hashCode() == y.hashCode()

Что означает согласованность?

Не надо делать так:

import java.util.Random;

public class Foo {
    private final Random random = new Random();

    @Override
    public int hashCode() {
        return random.nextInt();
    }

    @Override
    public boolean equals(Object o) {
        return random.nextBoolean();
    }
}
